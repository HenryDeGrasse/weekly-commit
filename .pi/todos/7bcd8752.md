## Concept
Run risk prediction on historical seed data weeks 1-8, compare predictions to actual outcomes in weeks 9-12.
Publish concrete precision/recall numbers.

## Implementation
- New test class: `HistoricalReplayBenchmark.java` (@Tag("eval"))
- Load historical plans from seed data
- For each week: compute risk signals as-of lock time
- Compare predictions to actual reconcile outcomes
- Compute: precision, recall, F1 for each risk type
- Output results to `build/eval-results/replay-benchmark.json`
- Summarize in `docs/eval-results.md`

## Metrics to track
- Carry-forward prediction accuracy
- Overcommit prediction vs actual completion ratio
- Risk signal false positive rate
- Overall prediction calibration
