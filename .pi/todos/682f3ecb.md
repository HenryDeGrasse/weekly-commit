## Concept
Given a draft plan + hypothetical changes, simulate the impact:
- Capacity impact (points over/under budget)
- RCDO coverage change (which branches gain/lose effort)
- Risk delta (carry-forward likelihood, overcommit risk)
- LLM narration of implications and recommendations

## Backend
- `WhatIfRequest` DTO with hypothetical mutations (add/remove/modify commits)
- `WhatIfService.java` — applies mutations to in-memory plan copy, runs rules engine + feature lookup
- `WhatIfResponse` DTO with impact analysis
- Endpoint: `POST /api/ai/what-if`

## Frontend
- What-If panel/dialog accessible from MyWeek page
- Show before/after capacity, RCDO coverage, risk signals
- LLM-generated narrative explaining implications

## Key: uses existing infrastructure
- RiskDetectionService rules (in-memory, no persistence)
- UserWeekFact/CarryForwardFact for historical context
- RcdoWeekRollup for coverage analysis
- LLM for narration only
