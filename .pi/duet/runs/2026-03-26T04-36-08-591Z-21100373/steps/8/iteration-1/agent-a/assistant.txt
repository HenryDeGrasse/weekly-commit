### `AiController.java` updated

Three new endpoints:
1. **`POST /api/ai/query`** — Calls `SemanticQueryService.query()`, maps `RagQueryResult` → `RagQueryResponse`; returns `aiAvailable: false` on any exception
2. **`GET /api/teams/{id}/week/{weekStart}/ai-insights`** — Queries `findByTeamIdAndWeekStartDateAndSuggestionType(..., "TEAM_INSIGHT")`, parses each `suggestionPayload` JSON into `InsightCardDto`
3. **`GET /api/plans/{id}/ai-insights`** — Queries `findByPlanIdAndSuggestionType(..., "PERSONAL_INSIGHT")`, same parsing

Private `parseInsightCards()` helper extracts `insightText`, `severity`, `actionSuggestion`, `sourceEntityIds` from the JSON payload and `createdAt` from the entity timestamp; unparseable rows are silently skipped.
